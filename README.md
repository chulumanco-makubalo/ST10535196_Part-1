## Features
### Password validation
Validates that the username contains an underscore and is no longer than 5 characters.
Enforces password complexity:
Minimum 8 characters.
At least 1 uppercase letter.
At least 1 number.
At least 1 special character.

---

### South African phone number validation
Validates phone numbers that start with +27
Ensures the correct number of digits after the country code
Rejects improperly formatted or incomplete numbers
Lightweight and easy to integrate into existing applications


---

### User Login Validation
Checks if the entered username, password, and phone number match registered credentials.
Returns a friendly login message upon success or failure.
Unit Testing
Unit tests to validate username, password, and phone number rules.
